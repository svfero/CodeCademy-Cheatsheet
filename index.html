CSS Styling Cheat Sheet

Display and Positioning

Tag

Name

Description
z-index: Z-index Property How far back or forward an element appears.
position: Position Property Controls where on the page an element appears.
display: Display Property Determines the type of block for an element.
float: Float Property Determines how far left or right an element should float within its parent element.
clear: Clear Property Specifies how an element should behave when it bumps into another element within the same containing element.
